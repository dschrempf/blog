Dateutils

<2015-01-07 Wed> Did you ever need to calculate the time difference between to consecutive time stamps from a log file or something similar? Check dateutils out, it is really useful: 

ddiff -i '%H:%M:%S' '19:09:43,683' '19:34:10,350'
